# EnglishTalk 🎙️

Aplicación web para practicar conversación en inglés con subtítulos en tiempo real en inglés y español.

## Características ✨

- **Reconocimiento de voz en tiempo real** - Habla en inglés y ve la transcripción instantáneamente
- **Traducción automática** - Traduce automáticamente al español usando API gratuita
- **Subtítulos duales** - Visualiza inglés y español simultáneamente
- **Historial de conversación** - Guarda tus últimas 10 frases con timestamps
- **Interfaz moderna y responsiva** - Diseño atractivo que funciona en desktop y móvil

## Cómo usar 🚀

1. Abre `index.html` en tu navegador (Chrome, Edge o Safari recomendados)
2. Haz click en "Start Conversation"
3. Permite el acceso al micrófono cuando el navegador lo solicite
4. ¡Comienza a hablar en inglés!
5. Verás la transcripción en inglés y la traducción al español en tiempo real

## Requisitos 📋

- Navegador moderno con soporte para Web Speech API (Chrome, Edge, Safari)
- Conexión a internet (para la API de traducción)
- Micrófono funcional

## Atajos de teclado ⌨️

- `Ctrl + Space` - Iniciar/Detener conversación

## Tecnologías 🛠️

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- JavaScript (ES6+)
- Web Speech API
- MyMemory Translation API

## Notas importantes ⚠️

- La aplicación funciona mejor en Chrome/Edge debido a su mejor soporte de Web Speech API
- Se requiere conexión a internet para la traducción
- El reconocimiento de voz puede variar según la calidad del micrófono y pronunciación
- La API de traducción es gratuita pero tiene límites de uso diario
